## 카드 뭉치

- [x] 코니는 다음과 같은 규칙으로 카드에 적힌 단어들을 사용해 원하는 순서의 단어 배열을 만들 수 있는지 알고 싶다.

  문자열로 이루어진 배열 cards1, cards2와 원하는 단어 배열 goal이 매개변수로 주어질 때, cards1과 cards2에 적힌 단어들로 goal를 만들 있다면 "Yes"를, 만들 수 없다면 "No"를 return하는 solution 함수를 완성하라.
  - 원하는 카드 뭉치에서 카드를 순서대로 한 장씩 사용합니다.
  - 한 번 사용한 카드는 다시 사용할 수 없습니다.
  - 카드를 사용하지 않고 다음 카드로 넘어갈 수 없습니다.
  - 기존에 주어진 카드 뭉치의 단어 순서는 바꿀 수 없습니다.
